1. 소개
    1) 구글이 만든 머신러닝을 위한 엔드투엔드 오픈소스 플랫폼 
        - 기계학습 프레임워크 
    2) GPU 연산의 CUDA(Computed Unified Device Architecture) 기술을
       지원함으로써 빠른 처리 가능 
        - GPU 연동을 통해 효율적으로 딥러닝 모델을 학습할수 있음.
    3) 2.x 버전
        - CPU, GPU 통합 플랫폼으로 제공 
        - 프로그램 소스 차원에서는 변경 없이 GPU 사용 가능 
    4) Tensor라는 다차원 데이터 구조 사용 
        - 텐서를 Numpy 배열처럼 사용할수 있음     

2. GPU 사용 여부 체크

3. 텐서 소개
    1) TensorFlow에서의 텐서(tensor)는 기능적으로 
       넘파이(Numpy)와 매우 유사함 
    2) 기본적으로 다차원 배열을 처리하기에 적합한 자료구조
    3) TensorFlow에서의 텐서(tensor)는 "자동미분" 기능 제공 

4. 텐서의 속성
    1) 기본 속성
        - 모양(shape)
        - 자료형(data type)
        - 저장된 장치 

5. 텐서 초기화 
    - 다른 텐서로부터 텐서 초기화 

6. 텐서의 형변환 및 차원 조작 
    1) 텐서는 넘파이(Numpy) 배열처럼 조작할수 있음 
    2) 텐서의 특정 차원 접근     

7. 텐서 이어붙이기 (concatenate)
    1) 두 텐서를 이어 붙여 연결하여 새로운 텐서 생성 

8. 텐서 형변환 (Type Casting)
    1) 텐서의 자료형(정수, 실수 등) 변환 가능     

9. 텐서의 모양 변경
    1) reshape()는 텐서의 모양 변경할때 사용 
    2) 이때, 텐서의 순서는 변경되지 않음 

10. 텐서의 차원 교환 
    1) 하나의 텐서에서 특정한 차원끼리 순서를 교체할수 있음
