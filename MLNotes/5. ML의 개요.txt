1. ML 정의
    1) 아서 사무엘(Arthur Samuel)
        - 기계가 일일이 코드로 명시하지 않은 동작을
          데이터로부터 학습하여 실행할 수 있도록 하는
          알고리즘을 개발하는 연구 분야
        - 컴퓨터에 명시적인 프로그램 없이 배울수 있는 능력을
          부여하는 연구 분야   

    2) 머신러닝은 정해진 프로그램을 수행하는 것이라기보다,
       사람이 학습하듯이 컴퓨터에도 데이터들을 줘서 학습하게
       함으로써 새로운 지식을 얻어내게 하는 분야 

    3) 기존 개발방식과의 차이 
    
                        *SW 프로그램 개발방식   

                연역적     로직흐름 설계(Flow Chart) 
                            - If, For....

                        *머신러닝 학습 방식             
      입력      귀납적      학습 알고리즘        ^   
      데이터                                    y = f(x)  

    4) 경험(Experience)을 통해 작업(Task)에 대한
       성능(Performance)을 향상하기 위한 
       알고리즘을 개발하는 분야   
        - 예) 스팸메일 필터링 프로그램 
            - E : 스팸인지 아닌지 분류된 이메일 
              T : 이메일을 스팸인지 아닌지 분류 
              P : 올바르게 분류된 메일의 수와 
                  잘못분류된 메일의 수를 기반으로 
                  계산된 분류정확도  

    5) 머신러닝은 입력 데이터를 기반으로 
       예측이나 결정을 위한 모델을 구축하는 방식을 취함 

    6) 인공신경망 알고리즘 
        - 뇌세포의 뉴런과 시냅스의 연결인 신경망을 통해
          입력 신호가 들어오면 뉴런이 활성화되는 과정을 모방 
        -   ^
            y = b1x + b0  (예측 모델링)    

    7) 단계로 정의
        - 학습 단계 
            - 반복적인 계산으로 학습하여 최적의 모델을 만듦
        - 예측 단계
            - 만들어진 모델을 운영환경에 배포하여 
              새로운 데이터의 예측값 추론 
        - 학습/재학습 단계
            - 학습기간과 데이터가 증가하며 학습 모델의
              개선을 반복 수행                                 

2. ML 기술 발전 배경
    1) 통계학(통계 분석)
        - 데이터 분석에 많은 이론적 배경 제공 
        - 가설 수립 및 검정 
        - 수치적 예측을 위한 예측통계 (회귀분석)   
    2) 데이터 마이닝 
        - 수많은 데이터에서 숨겨진 패턴을 찾기 위한
          컴퓨터 알고리즘이 발전한 기법 
        - 군집화, 분류, 예측, 분석 등  
        - 주체
            - data mining : 데이터 분석가가 가치를 추출함 
            - Machine Learning
                - 대상 : 빅데이터 
                - 주체 : 학습, 예측하는 시스템 
                         머신러닝 알고리즘 
    3) 인공신경망 
        - 인간의 두뇌를 모방한 인공신경망 (ANN, Perceptron)
        - 인공지능(AI)의 출발점                 

3. ML의 학습원리 
    1) 인간의 학습
        - 사냥
            - 동물의 행동 패턴 학습
            - 이기는 싸움의 패턴 학습 
        - 농사
            - 농작물이 자라는 패턴 학습
            - 계절의 변화에 대한 패턴 학습 
    2) 기계의 학습
        - 데이터에 숨겨진 패턴(새로운 지식) 발견    
        - 과거의 데이터를 학습하여 미래의 새로운 상황을 
          예측하기 위한 모델을 생성함 

        - X변수가 Y변수에 미치는 관계성을 반복적으로
          계산하여 함수의 형태로 만드는 과정
          - 함수 즉 모델의 가중치(파라미터)를 찾아가는 과정 
        
        - 학습은 실제 y값과 예측된 y값의 차이(오차)를
          줄이는 방향으로 모델 가중치 값의 업데이트를 
          반복 수행   
                                                      ^  
          X1, X2,.... Xn | Y  =>   반복학습         => y = f(bn * xn) + b0
          X1, X2,.... Xn | Y      (regression/tree)         --- 
                                                        모델 가중치 

                                                      x를 대입하여 y값을 예측(추론)

4. ML Workflow
  1) 머신러닝 알고리즘을 이용하여 문제를 해결하는 방법론,
     머신러닝 프로젝트 수행 방법

    - Module1 : 분석 문제 정의
    - Module2 : 데이터 수집 및 정제 
    - Module3 : 탐색적 데이터 분석(EDA)
       - 결측치 처리 
       - 데이터 시각화 
       - 통계량 분석  
    - Module4 : 피처 엔지니어링 
      - 차원의 변경
      - 상관관계 분석 
      - 피처 추출 
      - 범주형 변수 인코딩   
    - Module5    
      - 예측 모델 개발 및 평가
