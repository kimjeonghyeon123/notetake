1. Python3 standard type
    1) None
        - NoneType
    
    2) Numbers
        - int, float, bool

    3) Sequence
        - Mutable(변하는)
            - list(자유롭게 변경이 가능한 여러 개의 데이터 묶음)
        - Immutable(불변하는)
            - tuple (생성 후 변경이 불가능한 여러 개의 데이터 묶음)
            - str (문자열, character sequence)
    
    4) Set
        - set (집합)

    5) Mappings
        - dict (key-value store 방식을 가지고 자료를 저장하는 구조)

2. 기본 입출력
    1) print()를 이용한 표준 출력
        - print(값)
        - print(값1, 값2, ....)
            - 여러 개의 값을 공백을 기준으로 출력
            - end 속성
                - 마지막 문자열을 출력하고 출력할 문자
                - 초기 설정 : 줄바꿈
            - sep 속성
                - 여러 변수를 출력할 때, 각 값을 구분하기 위한 문자
                - 초기 설정 : 공백 문자
        
    2) input()을 이용한 표준 입력
        - 키보드(표준 입력장치)로부터 문자열을 입력 받음
        - 사용자에게 입력받은 정보를 정수 형태로 처리하기 위하여 int() 함수를 함께 사용할 수 있음

3. 정수 데이터
    1) 컴퓨터의 메모리가 허용하는 한, 정수 데이터의 크기 제한이 없다.

4. 실수 데이터
    1) 실수 데이터와 정수 데이터 사이에서 연산 수행 시, 데이터의 형 변환 (int -> double)

5. 사칙연산
    1) 사칙연산 : +, -, *, /, %
    2) 몫 연산자   : //
    3) 제곱 연산자 : **
