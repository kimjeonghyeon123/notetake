1. 프론트엔드 개발
    1) HTML, CSS, JS를 사용해 데이터를 GUI로 변환하고,
       그것으로 사용자와 상호 작용할 수 있도록 하는 것
    
    백엔드 ----------> 프론트엔드 ----------> 사용자
    
    2) HTML
        - 페이지의 제목, 문단, 표, 이미지, 동영상 등의 웹의 구조를 담당

    3) CSS
        - 실제 화면에 표시하는 방법을 지정해 콘텐츠를 꾸며주는 시각적인 표현(정적)
          을 담당
    
    4) JS
        - 콘텐츠를 바꾸고 움직이는 등 페이지를 동적 처리를 담당
