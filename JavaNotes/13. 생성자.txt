1. 생성자(constructor)
    1) 객체 생성할 때만 호출되는 특수한 메서드임
    2) new 연산자가 객체의 생성자를 이용해 객체를 생성해 줌
    3) 생성자 명은 클래스명과 같음
    4) 반환타입이 없음
    5) 정의하지 않으면 매개변수가 없는 생성자가 자동으로 만들어 짐 -> 디폴트 생성자
    6) 간혹 private으로 하는 경우도 있음
    7) 반드시 하나 이상 존재
    8)  기본 생성자
        - 정의하지 않아도 컴파일러가 클래스명만 보고 자동으로 만들어줌
        - 특별히 수행하는 기능 없음(구현부가 없음)

2. 생성자 만들기
    1) 컴파일러가 제공해주는 기본 생성자 외에 필요에 의해 생성자를 직접 구현할 수 있음
    2) 생성자 오버로딩 가능

3. 오버로딩 VS 오버라이딩
    1) 오버로딩
        - 메서드명은 같지만 파라미터 타입이나 개수가 다름

    2) 오버라이딩
        - 부모 클래스에서 사용한 메서드를 자식 클래스에서 재정의 해 쓸 수 있다.

4.
    1) private   : 누구나, 외부 어디서든 접근 가능
       public    : 같은 클래스 내부만 접근 가능
       protected : 부모 자식 간 가능
       default   : 같은 패키지 내부에서만 접근 가능

5. this
    1) 자기 자신을 가리키는 참조 변수
    2) new라는 연산자가 heap에 인스턴스에 할당할 때, 활성화 됨

    this()
        1) 생성자에서 다른 생성자 호출할 때 사용
