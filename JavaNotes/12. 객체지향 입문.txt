1. 객체와 클래스
    1) 클래스는 객체의 틀이다.
    2) 클래스는 객체를 프로그래밍에서 사용하려고 추상화한 것
    3) 클래스를 기초로 객체를 구현해서 메모리에 탑재해 메모리주소 부여
    4) 코드
        - Student 클래스의 객체 생성        ┌생성자
            - Student myStudent = new Student();
                                   └객체생성문법
        - new 키원드
            - 객체가 생성되면 참조형이 되어 동적메모리 할당 공간(Heap)에 내용을 사용할 수 있음

    6) this : 자기자신을 가리키는 포인터
       super : 조상객체의 주소
       Object : 모든 클래스의 부모(슈퍼) 클래스

    7) 인스턴스(instance)
        - 클래스는 객체의 속성을 정의하고, 기능을 구현하여 만들어 놓은 코드 상태
        - 각각 다른 멤버 변수 값을 가지게 됨
        - new 키워드 사용하여 인스턴스 생성
        - 동적 메모리(Heap memory)에 할당됨
2. 함수와 메서드
    1) 함수(function)
    2) 메서드(method)
        - 객체의 기능을 구현하기 위해 클래스 내부에 구현되는 함수
        - 멤버 함수 (member function)라고도 함
        
    3) 오버로딩(overloading)
        - 하나의 클래스 내에 메서드명은 같지만 매개변수 개수나 자료형은 다른 것
