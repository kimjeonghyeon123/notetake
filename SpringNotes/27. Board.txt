1. 게시물 목록과 페이징 처리
    1) 용어(변수)
        - pageSize
        - navi size
        - begin page
        - end page
        - page(전체)

        - offset
            - 게시물 맨 처음부터 얼마나 떨어져 있는지 나타내는 수
            - 각 페이지의 첫번째 게시물 번호로부터 떨어진 거리
                - page      offset
                 ------------------
                   1           0
                   2           10
                   3           20
                   4           30

        - row_count
            - (앞으로)읽어올 row의 수
    
    2) URI 정의
                          URL              HTTP 메소드
        1) 읽기    /board/read?bno=번호         GET         지정된 번호의 게시물
        2) 삭제    /board/remove                POST       게시물 삭제

        3) 쓰기    /board/write                 GET         게시물 작성할 빈 화면 보여줌
                   /board/write                 POST        작성한 게시물 보여줌
        4) 수정     /board/modify?bno=번호      GET         게시물 수정하기 위해 수정화면 보여줌
                    /board/modify               POST       수정된 게시물 저장
