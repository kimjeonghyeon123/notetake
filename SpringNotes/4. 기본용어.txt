1. 서버(Server)
    1) 웹에서 서비스를 제공하는 컴퓨터 시스템.
    2) 서비스를 제공하는 웹 애플리케이션은 서버 컴퓨터에 
       파일의 형태로 저장되어 있음  

2. 웹 서버(Web Server)       
    1) 사용자로부터 HTTP를 통해 요청을 받거나, 
       웹 컨테이너가 전달해 준 결과물을 정적인 페이지로 생성하여
       사용자에게 응답해주는 소프트웨어 
    2) 웹 페이지는 주로 HTML, CSS, JS 등으로 구성됨.  

3. 웹 컨테이너 (Web Container)
    1) 웹 서버가 전송해준 요청을 기초로 동적인 페이지를 생성하여
       웹 서버로 돌려줌 
    2) 동적인 페이지
        - 사용자마다 다른 결과로 응답할 수 있기 때문임.
        - 간단한 예로 로그인이 있음 
            - 로그인한 아이디를 보고 각자에 맞는 페이지를 구성해 
              보내줌                 

4. WAS (Web Application Server)           
    1) 웹 애플리케이션이 실행될 수 있는 환경을 제공하는 소프트웨어
        - 컴퓨터에서 운영체제와 비슷한 역할을 하는 소프트웨어
    2) 웹 서버와 웹 컨테이너를 포함한 개념임 
    3) 예: 톰캣(Tomcat), 웹로직, 웹스피어 등         

5. 포트 (Port)    
    1) 항구 
    2) 컴퓨터 사이에서 데이터를 주고 받을 수 있는 통로 
    3) 인터넷에서는 IP 주소를 통해 서버 컴퓨터의 위치를 파악함
    4) 그런 다음 그 컴퓨터가 제공하는 특정 서비스는 포트 번호를 통해
       알수 있음  
    5) 대표적으로 HTTP는 80번 포트를, 보안이 적용된 HTTPS는 443번 
       포트를 사용함        

6. 정적 웹 페이지 (static web page)
    1) 저장되어 있는 파일을 그대로 웹 브라우저에 전송해 출력하는
       가장 기본적인 웹 페이지를 말함.
    2) 같은 페이지는 항상 동일한 모습을 보여주기 때문에 '정적'이라함  
    3) HTML, JS, CSS, 이미지 등 

7. 동적 웹 페이지 (dynamic web page)
    1) 동일한 페이지 할지라도 그때그때 내용이 달라질 수 있는 
       웹 페이지임.
    2) 서버가 클라이언트의 요청을 해석하여 가장 적절한 웹 페이지를 
       그때그때 생성해 보여주는 기술임 
    3) 클라이언트는 요청한 계정, 시간, 지역, 언어, 기타 입력값에 
       따라 다른 결과를 받음
