1. 기술 스택 
    1) 구현에 필요한 기술 구성 
    2) 기술 선택 기준은 가장 범용적인 기술 사용 
    3) 학습 곡선이 낮도록 선택 
    4) 범용적인 Frontend 기술과 Backend 기술을 선정 

2. Frontend(프론트엔드)
    1) 템플릿 엔진 
        - Thymeleaf Template 
            - Spring Boot에서 공식적으로 제공하는 템플릿 엔진
    2) UI 템플릿   
        - Bootstrap
            - 오픈소스, 범용적인 서비스 UI 템플릿 
    3) 자바스크립트 라이브러리 
        - jQuery
            - 쉽게 사용 가능한 JavaScript Library 

3. Backend (백엔드)
    1) Language
        - Java 11
            - 범용적인 애플리케이션 개발 언어 
    2) Framework 
        - Spring Boot 2.7.x 
            - 범용적인 자바 애플리케이션 플랫폼 
            - 톰캣 서버가 내장되어 별도 서버가 필요하지 않음 
    3) Database 
        - Postgresql 14 (MySQL 8.x)
            - 범용적인 데이터 관리 솔류션                         

4. Thymeleaf 
    1) 화면 개발 템플릿 
    2) 기본 문법 
        - th:text 
            - 기본적인 출력 
            - 출력 예시 
                - 문자열 출력
                    <span th:text="${}"></span>
                - 문자열 결합 출력    
                    <span th:text="|${} 추가 문자열|"></span>

    3) 레이아웃 문법 
        - 화면을 여러 맥락에 따라 나눌 수 있음 
        - 반복되는 화면 영역을 별도로 분리하여 여러 다른 페이지에도 
          적용할 수 있음 

        - 헤더
        - 푸터 
        - 메뉴 
        - 상단 검색창 
        - 기타 공통 영역 

        - 의존성 추가 : thymeleaf-layout-dialect

        - 관련 문법
            - th:fragment=""   프레그먼트 선언
